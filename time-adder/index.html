Time Adder

Description +

This page allows you to enter time and unit of time to get the result.
The unit of the final result is based on whether the final time is a whole number of days or hours or minutes. Otherwise final unit wll be seconds.

Rules:

  1. Time should be an integer greater than or equal to 0
  2. Allowed units are ["day", "days", "hour", "hours", "minute", "minutes", "second", "seconds"]
  3. Please enter appropriate units. E.g if time = 1, then use singular units otherwise use plural units

Examples:

Inputs Result

1. 10 seconds 20 minutes 1210 seconds
2. 30 seconds 20 days 1728030 seconds
3. 5 minutes 1 hour 65 minutes
4. 0 days 0 minutes 0 seconds
